JavaScript Loops Activity

Activity Instructions

Open your browser's Developer Tools (F12) and go to the Console tab. Copy and paste the following code snippets to test them:

Variable Declarations

const DAYS = 6;
const LIMIT = 30;
let studentReport = [11, 42, 33, 64, 29, 37, 44];

1. For Loop - Print values below 30

// for loop
for (let i = 0; i < studentReport.length; i++) {
  if (studentReport[i] < LIMIT) {
    console.log(studentReport[i]);
  }
}

2. While Loop - Print values below 30

// while loop
let i = 0;
while (i < studentReport.length) {
  if (studentReport[i] < LIMIT) {
    console.log(studentReport[i]);
  }
  i++;
}

3. ForEach Loop - Print values below 30

// forEach loop
studentReport.forEach(function (item) {
  if (item < LIMIT) {
    console.log(item);
  }
});

4. For...In Loop - Print values below 30

// for...in loop
for (let i in studentReport) {
  if (studentReport[i] < LIMIT) {
    console.log(studentReport[i]);
  }
}

5. Generate Day Names for Next DAYS Days

Use any type of loop to dynamically produce the day names starting from today.

// Example using for loop to generate future day names
const dayNames = ['Sunday', 'Monday', 'Tuesday', 'Wednesday', 'Thursday', 'Friday', 'Saturday'];
const today = new Date();
const currentDayIndex = today.getDay();

console.log('Next ' + DAYS + ' days starting tomorrow:');
for (let i = 1; i <= DAYS; i++) {
  const futureDayIndex = (currentDayIndex + i) % 7;
  console.log(dayNames[futureDayIndex]);
}
